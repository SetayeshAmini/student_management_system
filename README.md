# Student Management System

A Python-based Object-Oriented Programming (OOP) application designed to manage student records, courses, grades, assessments, and quizzes.

## Features

- Gradebook Management: Calculate letters and percentages dynamically.
- Advanced Search: Search for students by their ID or Name (case-insensitive).
- Record Operations: Add, update, or delete student profiles and scores cleanly.
- Modular Design: Separated logically into distinct modules (`Student`, Course, Gradebook, Quiz, Exam, Project, etc.).

## Project Structure

The project consists of the following source files:
- main.py - The entry point of the application.
- Student.py - Defines the Student entity and properties.
- Course.py - Manages course details and enrollments.
- Gradebook.py - Handles calculation logic ( search_students, etc.).
- Assessment.py, Quiz.py, Exam.py, Project.py - Manage various evaluation components.

## How to Run

1. Clone this repository:
     git clone https://github.com
   2. Navigate to the project directory:
     cd student_management_system
   3. Run the application:
     python main.py
